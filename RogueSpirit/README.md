## 📖 Описание

**Rogue Spirit** - это динамичный 2D экшен-рогалик в стиле Soul Knight с процедурной генерацией уровней, разнообразным оружием, системой артефактов и эпичными босс-файтами.

### ✨ Особенности игры:
- 🗡️ **15+ видов оружия**: от мечей до ракетниц
- 🎭 **8 уникальных персонажей** с различными способностями
- 🌍 **5 биомов**: Лес, Долина, Горы, Вода, Вулкан
- 👾 **40+ типов врагов** и 5 эпичных боссов
- 🎨 **Красивая пиксельная графика** с анимациями
- 🎵 **Атмосферный саундтрек** и звуковые эффекты
- 🏆 **Система достижений** и прогрессии

## 🌟 Игровые Уровни

### Уровень 1: Зачарованный Лес 🌲
- **Тема**: Мистический лес с древними деревьями
- **Враги**: Гоблины, пауки, волки, духи леса
- **Босс**: Древний Треант Хранитель
- **Особенности**: Падающие ветки, ядовитые грибы

### Уровень 2: Тёмная Долина 🌑
- **Тема**: Мрачное ущелье с сильными ветрами
- **Враги**: Бандиты, скелеты, теневые охотники
- **Босс**: Повелитель Штормов
- **Особенности**: Молнии, падающие камни, сильный ветер

### Уровень 3: Ледяные Горы ❄️
- **Тема**: Заснеженные вершины с метелью
- **Враги**: Ледяные гоблины, йети, ледяные элементали
- **Босс**: Ледяной Дракон
- **Особенности**: Скользкий лёд, лавины, обледенение

### Уровень 4: Мистические Воды 🌊
- **Тема**: Водный храм с озёрами и реками
- **Враги**: Пираньи, крабы, медузы, русалки
- **Босс**: Левиафан
- **Особенности**: Подводные секции, кислород, течения

### Уровень 5: Огненный Вулкан 🌋
- **Тема**: Активный вулкан с потоками лавы
- **Враги**: Огненные импы, лавовые големы, демоны
- **Босс**: Бог Вулкана (Магма Титан)
- **Особенности**: Лава, извержения, падающие метеоры

## 🚀 Быстрый Старт

### Требования:
- **Unity**: 6000.4.0f1 или новее
- **Visual Studio**: 2022 или 2026
- **OS**: Windows 10/11, macOS, Linux
- **RAM**: 8 GB минимум
- **Storage**: 2 GB свободного места

### Установка:

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/yourusername/RogueSpirit.git
cd RogueSpirit
```

2. **Откройте проект в Unity:**
   - Запустите Unity Hub
   - Нажмите "Add" → выберите папку проекта
   - Откройте проект в Unity 6000.4.0f1

3. **Откройте в Visual Studio:**
   - В Unity: `Edit` → `Preferences` → `External Tools`
   - Выберите Visual Studio 2022/2026
   - Дважды кликните на любой скрипт для открытия

4. **Запустите игру:**
   - В Unity нажмите `Play` (▶)
   - Или постройте билд: `File` → `Build Settings` → `Build`

## 🎯 Геймплей

### Система Здоровья:
- ♥ **Красные сердца** - текущее здоровье
- ♡ **Пустые сердца** - максимальное здоровье
- При потере всех сердец - **GAME OVER**
- **Зелья здоровья** восстанавливают сердца

### Система Очков:
- Убийство врагов: +10-100 очков
- Убийство босса: +1000 очков
- Комбо множитель: x1.0 - x5.0
- Достижение определённых очков открывает **скиллы**

### Артефакты:
- **Огненный кристалл**: +15% урона огнём
- **Ледяной осколок**: шанс заморозить врага
- **Сапоги скорости**: +25% скорости движения
- **Клык вампира**: лечение при убийстве

## 🛠️ Разработка

### Добавление нового оружия:

```csharp
public class NewWeapon : RangedWeapon
{
    void Start()
    {
        damage = 25f;
        fireRate = 5f;
        ammoCapacity = 30;
    }
}
```

### Создание нового врага:

1. Создайте скрипт в `Assets/Scripts/Enemies/`
2. Наследуйтесь от `Enemy` класса
3. Настройте AI поведение
4. Создайте префаб и добавьте в `SpawnManager`

### Добавление уровня:

1. Создайте новую сцену в `Assets/Scenes/`
2. Настройте тайлсет и фон
3. Добавьте спавнеры врагов
4. Зарегистрируйте в `LevelManager`
