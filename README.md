## 📖 Описание игры

**Rogue Spirit** — это динамичный 2D экшен-платформер в стиле roguelike, вдохновленный культовой игрой *Soul Knight*. Погрузитесь в захватывающее приключение, где вам предстоит пройти через 5 уникальных миров, каждый со своей атмосферой, врагами и эпическими боссами.

Управляйте героем, исследуйте процедурно генерируемые уровни, собирайте мощное оружие и артефакты, сражайтесь с полчищами врагов и станьте легендой!

### ✨ Ключевые особенности

- 🏃 **Динамичное управление**: Бег, прыжки, приседания и точное движение
- ⚔️ **Арсенал оружия**: От мечей до магических посохов — более 50 видов оружия
- 🌍 **5 уникальных миров**: Каждый уровень имеет свою тематику, визуальный стиль и саундтрек
- 👹 **Разнообразные враги**: Более 30 типов противников, соответствующих каждому миру
- 🐉 **Эпические боссы**: Уникальные боссы в конце каждого уровня с собственными механиками
- 💎 **Система артефактов**: Собирайте реликвии для усиления способностей героя
- ❤️ **Система здоровья**: Классическая система сердец с возможностью лечения
- 🏆 **Система очков**: Зарабатывайте очки за победы и открывайте новые навыки
- 🎨 **Кастомизация**: Выбор скина героя перед началом игры
- 🔄 **Прокачка**: Улучшайте характеристики и открывайте новые способности

## 🎯 Геймплей

### Управление

| Клавиша | Действие |
|---------|----------|
| **A** | Движение влево / назад |
| **D** | Движение вправо / вперед |
| **W** | Прыжок |
| **S** | Приседание |
| **Shift** | Бег (ускорение) |
| **J** | Атака / Использование оружия |
| **K** | Использование артефакта |
| **L** | Взаимодействие с объектами |
| **Esc** | Пауза / Меню |

### Механики игры

#### 🏃 Перемещение
Герой может свободно перемещаться по уровню, используя платформерную механику:
- **Бег**: Зажмите Shift для ускорения
- **Прыжок**: Одинарный прыжок с возможностью двойного (при открытии навыка)
- **Приседание**: Позволяет проходить под низкими препятствиями и уклоняться от атак
- **Лестницы**: Автоматическое поднятие и спуск по ступенькам

#### ⚔️ Боевая система
- **Ближний бой**: Мечи, кинжалы, копья
- **Дальний бой**: Луки, арбалеты, огнестрельное оружие
- **Магия**: Посохы, свитки, магические сферы
- **Особые оружия**: Гранатометы, лазеры, бумеранги

#### 💎 Артефакты
Собирайте уникальные артефакты для получения специальных способностей:
- **Щит духа**: Временная неуязвимость
- **Крылья ветра**: Двойной прыжок
- **Сердце дракона**: Дополнительное сердце здоровья
- **Кольцо скорости**: Увеличение скорости передвижения
- **Амулет урона**: Повышение урона на 25%

## 🌍 Уровни и Миры

### Уровень 1: 🌲 Дремучий Лес
*Первый шаг в опасное приключение*

- **Атмосфера**: Таинственный лес с густой растительностью
- **Визуальный стиль**: Зеленые тона, солнечные лучи сквозь кроны деревьев
- **Враги**:
  - 🐺 Дикие волки
  - 🍄 Грибные существа
  - 🦇 Летучие мыши
  - 🌿 Лесные духи
- **Босс**: 🌳 Древний Энт — гигантское дерево с мощными атаками ветвями
- **Особенности**: Подвижные платформы-ветки, скрытые проходы в дуплах

### Уровень 2: 🌑 Темная Долина
*Земля вечных ветров и теней*

- **Атмосфера**: Мрачная долина с сильными порывами ветра
- **Визуальный стиль**: Темно-фиолетовые и серые тона, динамическое освещение
- **Враги**:
  - 💀 Скелеты-воины
  - 👻 Призраки
  - 🦅 Ветрокрылые гарпии
  - ⚫ Теневые ассасины
- **Босс**: 🌪️ Повелитель Штормов — элементаль ветра с атаками торнадо
- **Особенности**: Ветровые потоки, влияющие на прыжки, подвижные облачные платформы

### Уровень 3: 🏔️ Снежные Горы
*Царство льда и снежных бурь*

- **Атмосфера**: Холодные вершины с постоянными снегопадами
- **Визуальный стиль**: Бело-голубая палитра, эффекты снежной бури
- **Враги**:
  - ❄️ Ледяные големы
  - 🐧 Пингвины-берсерки
  - 🦅 Снежные ястребы
  - ☃️ Злые снеговики
- **Босс**: 🧊 Ледяной Дракон — древний дракон с ледяным дыханием
- **Особенности**: Скользкие поверхности, лавины, ледяные шипы

### Уровень 4: 🌊 Водное Царство
*Тайны глубин и водной стихии*

- **Атмосфера**: Загадочное озеро с кувшинками и подводными пещерами
- **Визуальный стиль**: Синие и бирюзовые оттенки, эффекты преломления воды
- **Враги**:
  - 🐙 Щупальца глубин
  - 🐟 Рыбы-пираньи
  - 🐢 Бронированные черепахи
  - 🧜‍♀️ Водяные нимфы
- **Босс**: 🦑 Кракен-Хранитель — гигантский осьминог с множеством щупалец
- **Особенности**: Подводные зоны, кувшинки-платформы, водные течения

### Уровень 5: 🌋 Огненное Пекло
*Финальное испытание в жерле вулкана*

- **Атмосфера**: Адская жара, лава и извергающиеся вулканы
- **Визуальный стиль**: Оранжево-красная гамма, частицы огня и пепла
- **Враги**:
  - 🔥 Огненные элементали
  - 🦎 Лавовые ящеры
  - 💣 Взрывные демоны
  - 🐕 Адские гончие
- **Босс**: 🔥 Владыка Пламени — демонический существо с огненным мечом
- **Особенности**: Лава как опасная зона, извержения вулканов, огненные столбы

## 🎨 Система персонажей

### Выбор скина
Перед началом игры вы можете выбрать один из доступных скинов героя:

- **⚔️ Классический рыцарь**: Сбалансированный воин в доспехах
- **🏹 Лесной следопыт**: Мастер лука в зеленом камуфляже
- **🔥 Огненный маг**: Повелитель стихии огня в красных одеждах
- **❄️ Ледяная ведьма**: Хозяйка холода в синих одеяниях
- **💀 Темный странник**: Загадочный герой в черном плаще
- **⚡ Молниеносный ниндзя**: Быстрый ассасин с элементами молний

Каждый скин имеет уникальные визуальные эффекты, но одинаковые базовые характеристики для честного геймплея.

### Система здоровья
- ❤️ **Стартовое здоровье**: 3 сердца
- 💔 **Потеря здоровья**: При получении урона отнимается одно сердце
- 💖 **Лечение**: 
  - Зелья здоровья (восстанавливают 1 сердце)
  - Сердца-артефакты (добавляют временное сердце)
  - Фонтаны жизни (полное восстановление)
- ☠️ **Game Over**: При потере всех сердец игра заканчивается

### Система очков и прокачки
- 🎯 **Получение очков**: 
  - Убийство обычных врагов: 10-50 очков
  - Убийство элитных врагов: 100-200 очков
  - Победа над боссом: 500-1000 очков
- 🏆 **Магазин навыков**: При достижении определенных порогов очков можно приобрести:
  - Двойной прыжок (500 очков)
  - Увеличение макс. здоровья (750 очков)
  - Критический удар (1000 очков)
  - Вампиризм (1250 очков)
  - Неуязвимость на 3 сек (1500 очков)

- **Оперативная память**: 8 GB RAM
- **Видеокарта**: NVIDIA GeForce GTX 960 / AMD Radeon R9 280
- **Место на диске**: 2 GB (SSD рекомендуется)

### Технологии

- **Игровой движок**: Unity 2021.3 LTS или новее

- **Анимации**: Sprite Animation с использованием Animator Controller
- **Физика**: Unity Physics 2D
- **Звук**: FMOD Studio / Unity Audio System
- **Сохранение данных**: JSON / PlayerPrefs

## 🚀 Установка и запуск

### Шаг 1: Клонирование репозитория
```bash
git clone https://github.com/yourusername/RogueSpirit.git
cd RogueSpirit
```

### Шаг 2: Открытие в Unity
1. Запустите Unity Hub
2. Нажмите "Add" и выберите папку проекта
3. Откройте проект в Unity 2021.3 LTS или новее

### Шаг 3: Настройка
1. Дождитесь импорта всех ассетов
2. Проверьте настройки проекта в `Edit > Project Settings`
3. Убедитесь, что установлен URP (Universal Render Pipeline)

### Шаг 4: Сборка
1. Перейдите в `File > Build Settings`
2. Добавьте сцены: `MainMenu`, `GameScene`
3. Выберите платформу (Windows/Mac/Linux)
4. Нажмите "Build"

### Шаг 5: Запуск
Запустите созданный `.exe` файл (для Windows) или соответствующий исполняемый файл для вашей ОС.
